¿Que es html?

HTML (HyperText Markup Language) es el lenguaje estándar para estructurar y presentar contenido en páginas web.
Usa etiquetas para organizar elementos como texto, imágenes y enlaces, definiendo la estructura de una página.


Caracteristicas de HTML.

HTML te permite crear paginas web utilizando sus tags los cuales te permiten incluir imagenes, videos, sonidos ,enlaces y style
la cual se utiliza para darle un aspecto tanto al diseño de la pagina como al tipo de letra.

Formato

Para crear una pagina web utilizando html puedes hacerlo abriendo una pagina que tenga html como lenguaje de marcado
o simplemente ocupar un bloc de notas.esto puedes hacerlo siempre cuando coloques el nombre del documento como html
(ej:documento1.html) de esa manera se creara un directorio que dara a la pagina web creada.


Para crear la pagina web se utiliza el siguiente formato:

<!DOCTYPE html>
<html>
<head>
<title> </title>
</head>
<body>
<p> </p>
</body>
</html>


<DOCTYPE html> : Indica al navegador que la página está escrita en HTML5, la versión más reciente del lenguaje HTML.
Esta declaración permite que el navegador interprete y renderice correctamente el contenido según las especificaciones de HTML5.

<html> : Este es el elemento raíz de un documento HTML y envuelve todo el contenido de la página web.
Todo lo que contiene la página web debe estar dentro de esta etiqueta para que el navegador lo interprete como parte del documento HTML.

<head> : Sirve para proporcionar información esencial sobre el documento HTML y gestionar cómo se comportará o será interpretado por el
navegador. No muestra contenido visual, pero es clave para: <title>, <meta>, <link>, <script>, entre otros.

<title> : Define el título de la página web, que aparece en la pestaña del navegador, resultados de búsqueda y marcadores.

<body> : Define el contenido principal de la página web que se muestra en el navegador. Todo lo que el usuario ve e interactúa está dentro de esta etiqueta.

<p> : es utilizada para definir un párrafo de texto. Es una de las etiquetas más comunes para estructurar contenido textual en una página web.


Conclusion

Como se puede ver los principales actores al crear una pagina web con HTML son <DOCTYPE html>, <html>, <head> y <body>.
Siendo <DOCTYPE html> para definir la version, <html> para aclarar el lenguaje para la pagina, <head> para la informacion esencial de la pagina
y <body> para crear el contyenido dentro de la pagina.

Es importante aunque no siempre que cuando abras una etiqueta (<algo>), cerrarla utilizando la misma etiqueta pero con un / seguido del primer < (</algo>).
Tambien debes de cuidar tu ortografia pues html crea la pagina sin importar los errores por lo que es comun que la pagina omita un texto o algun contenido importante de la pagina.
