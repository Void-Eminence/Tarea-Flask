Imagenes

Para colocar imagenes solo debes utilizar:

< img src= >

Utilizando esta etiqueta puedes colocar una imagen en tu pagina web.solo debes mencionar la imagen utilizando ""
ejemplo:<img src="foto.jpg">. En caso de que no aparezca puedes utilizar / para mencionarla desde la carpeta origen.
ej:<img src="capeta1/foto.jpg">. En otro caso de que tu imagen sea de otro sitio web solo debes hacer mencion a esta.
ej:< img src="http://www.link de la img.com">.

Configuraciones extras:

< img src="" width="" height="" >: width y height se utilizan para darle un cierto tamaño a la imagen siendo height
aquel que controla el alto y width lo largo estos se controlan con numeros.

< img src="" alt="" >: alt proporciona un texto alternativo para las imágenes. Es una práctica esencial para mejorar
la accesibilidad, la usabilidad y la optimización

< img src="" title="" >: title en img muestra un texto emergente al pasar el cursor sobre la imagen.

Videos

Para ubicar un video en tu pagian web se puden utilizar las etiqueta:

<video src="">
-----o--------
<video>
<source src="" type="">
</video>
-----o----- <iframe src=""
</iframe>

Siendo la opcion 2 la mejor debido a su configuracion mas precisa.

<video> src="": La etiqueta video le dice a la pagina que tipo de añadido se implementara a la pagina,
mientras que src="", vincula el video elegido a la paguina ya sea este de otra pagina web o tuyo.

<source src="" type="">: source proporcionar diferentes fuentes del contenido.
Esto permite que el navegador elija automáticamente la fuente compatible que pueda reproducir.
src=le dice cual es el video, mientras que type indica el formato del video. (mp4,ogg,entre otros.)

Otras configuraciones:

<video width="" height="">: Ajusta el tamaño del video.

<video controls>: Añade los botones pausa, play, volumen,etc.

<video autoplay>: Inicia el video apenas ingreses a la pagina.

<video loop>: El video se reinicia(repite) cuando este acaba.

<video mute>: Comienza el video en silencio.

<video poster>: Ingresa la imagen que se mostrara antes de que el video se reproduzca.

Puedes colocar todos estas etiquetas en <video>.

Audios

Para añadir audio debes ocupar la siguiente etiqueta:

<audio src="">
----o ----
<audio> < source src="" type="" >

Como se puede ver audio ocupa la misma configuracion que video pero solo cambia la etiqueta img por audio.

Otras configuraciones:

Se pueden añadir todas las configuraciones de video.

  • width="" height=""
  • controls
  • autoplay
  • loop
  • mute
